# 開発メモ

## ローカル確認の起動手順

開発中は `index.html` を `file://` で直接開かず、ローカルサーバー経由で確認します。

```bash
cd /Users/momo/Desktop/家計簿アプリ
python3 -m http.server 5179
```

ブラウザで開くURL:

```text
http://127.0.0.1:5179/
```

## 保存確認の手順

1. `http://127.0.0.1:5179/` を開く
2. 収入、共有財布の金額、支出を入力する
3. ページをリロードする
4. 入力した金額、支出履歴、カテゴリ別、日別支出が残っていることを確認する
5. 月を切り替えて、その月のデータが残っていることを確認する
6. バックアップ画面が開けることを確認する

## 注意

- 保存確認、リロード確認、動作確認は `file://` ではなく `http://127.0.0.1:5179/` で行います。
- GitHub Pages 上でも、同じ保存キーの localStorage を正規保存として使います。
- `file://` での挙動はブラウザ環境に左右されるため、開発中の判定基準にしません。
